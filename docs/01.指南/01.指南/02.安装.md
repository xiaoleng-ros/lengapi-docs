- **JDK 17+**

## SpringBoot

### 🍊Maven
在项目的pom.xml的dependencies中加入以下内容:
```xml
<dependency>
    <groupId>com.leng</groupId>
    <artifactId>lengapi-client-sdk</artifactId>
    <version>0.0.5</version>
</dependency>
```
### 🍐Gradle
```
implementation group: 'com.leng', name: 'lengapi-client-sdk', version: '0.0.5'
```
